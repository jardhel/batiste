# Data Subject Request Log

**Owner:** DPO · **Version:** 0.1.0 · **Last reviewed:** 2026-04-20
**Maps to:** GDPR Arts. 12-22; CCPA §1798.100-.199.

> This log tracks every data-subject request received by Cachola Tech in the controller role or forwarded to us by a customer in the processor role. Private details are **redacted** from the shared copy of this register; the unredacted register lives in the DPO's controlled folder.

## How to use

On receipt of a request, copy the template below, assign the next ID, and complete through the "Decision" row within 1 working day. Drive the remaining rows to completion within the statutory deadline.

## Template

```
## DSR-YYYYMMDD-NN

- Received (UTC): 
- Channel: (email | portal | mail | customer-forward)
- Requester (redacted): initials only
- Role: controller / processor
- Right invoked: (15 access | 16 rectification | 17 erasure | 18 restriction | 20 portability | 21 object | 22 automated decision)
- Identity verified: yes/no, method
- Legal ground / exception: 
- Decision: grant / partial / refuse (with reasoning)
- Deadline: received + 30 days (+60 days extension if complex)
- Actions taken: 
- Evidence: (ledger seq, redaction key ID, export file)
- Response sent (UTC): 
- Outcome confirmed by requester: yes/no
- Closed (UTC): 
```
