## 2. Object of the declaration

Batiste is distributed as a set of software packages (TypeScript monorepo) for on-prem, air-gapped deployment. The declaration covers the software artefacts signed under release tag `v1.0.0`, including:

- `@batiste-aidk/core`, `@batiste-aidk/transport`, `@batiste-aidk/connectors`, `@batiste-aidk/code`, `@batiste-aidk/audit`, `@batiste-aidk/auth`, `@batiste-aidk/scope`, `@batiste-aidk/web`, `@batiste-aidk/aidk`, `@batiste-aidk/cli`, `@batiste-aidk/marketplace`.

## 3. Applicable EU legislation

This declaration asserts compliance with the following, to the extent Batiste is in scope for each as a component supplier:

- **Regulation (EU) 2016/679** (GDPR) — processor and controller obligations, Arts. 5, 25, 28, 30, 32.
- **Regulation (EU) 2022/2554** (DORA) — supplier requirements relevant to financial-entity customers, Arts. 28-30 where applicable.
- **Directive (EU) 2022/2555** (NIS2) — supplier security measures, Art. 21(2)(d)-(e).
- **Regulation (EU) 2024/1689** (AI Act) — technical documentation obligations for component suppliers to high-risk AI systems, Annex IV.
- **Regulation (EU) 2023/988** (General Product Safety Regulation) — where applicable.

Batiste itself is not a high-risk AI system listed in Annex III. This declaration is made to support customers who integrate Batiste into high-risk AI systems and need evidence of the component's properties.
